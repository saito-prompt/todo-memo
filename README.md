# やることメモ帳（ToDo App）

JavaScriptを学びながら作成した、シンプルなやることメモアプリです。

## 🔧 機能一覧

- 「やること」を追加・完了・削除できます
- Enterキーでタスクを追加可能
- 作成時刻が自動で表示されます
- 完了したタスクは下のリストに移動
- デザインはCSSで調整（削除・完了ボタンも視覚的にわかりやすく）

👉 [アプリを見る](https://saito-prompt.github.io/todo-memo/)
